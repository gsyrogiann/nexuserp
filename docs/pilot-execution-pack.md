# NexusERP Pilot Execution Pack

## Purpose

This document turns pilot readiness into an executable operating model.

## Pilot Structure

Each pilot should have:

* one named customer owner
* one named internal technical owner
* one named internal product owner
* one written success definition
* one agreed review cadence

## Pilot Phases

### Phase 1: Qualification

Confirm:

* the customer matches the ICP
* the workflows are close to current NexusERP scope
* pilot expectations are realistic
* the customer accepts active collaboration during the pilot

### Phase 2: Setup

Prepare:

* environment and secrets
* required integrations
* starter or migrated data
* known limitations briefing
* support and escalation contacts

### Phase 3: Live Validation

Track weekly:

* login and access stability
* customer and supplier workflow usage
* invoice workflow success
* Gmail/VoIP integration stability where applicable
* high-friction issues and workarounds

### Phase 4: Outcome Review

At pilot close, record:

* continue / expand / stop recommendation
* blockers discovered
* product gaps discovered
* required pricing or packaging adjustments
* reference potential for later selling

## Weekly Pilot Review Template

Record at least:

* customer name
* week number
* active users this week
* workflows used
* blockers
* regressions
* support incidents
* next commitments

## Pilot Success Signals

Positive signals include:

* the customer uses at least one core workflow daily
* the customer can complete invoicing and directory tasks without hand-holding
* feedback shifts from bugs to improvement requests
* integrations help rather than destabilize daily work

## Pilot Failure Signals

Treat the following as serious warnings:

* repeated access or data-trust complaints
* unresolved critical workflow regressions
* customer stops using the product after onboarding
* pilot owner becomes unresponsive because trust has dropped

## Required Artifacts Per Pilot

For each pilot, maintain:

* kickoff notes
* success criteria
* issue log
* weekly review notes
* final recommendation
